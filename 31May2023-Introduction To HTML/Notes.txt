Introduction to HTML:

HTML stands for HyperText Markup Language, used to describe web pages on the browser. 

Mark Up Language: It is a collection of tags that represent the content. 

Tag: A tag is a keyword that is enclosed in angular brackets. 

 examples:<html>, <head>, <body>, <p>......


HTML Tags: 

HTML provides plenty of tags that can be used to define / describe the content on the UI. 

ex: <img>, <a>,<input>,<textarea>....

Tags are not case sensitive which means we can use either upper case or lower case. however, Industry uses 
lower case only. 


HTML Document Structure:

     <head>
     
      </head>

      <body>

      </body>

q1) How does a web browser executes a html page?

When a web page (index.html) is given to web browser, It creates a tree called 
DOM (Document Object Model) tree, then the render engine uses the DOM tree and displays the content on the UI.
